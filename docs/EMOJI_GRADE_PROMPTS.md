## Unified Emoji-Grade Characteristics

### Scale Requirements
- **Minimum:** 16Ã—16px legible
- **Optimal:** 32Ã—32px with detail
- **Maximum:** 64Ã—64px with full composition

### Visual Simplification Rules
1. **Max 5 discrete elements** (human can count at glance)
2. **2-3 primary colors** + 1 accent
3. **Bold outlines** (1-2px at 32px scale)
4. **High contrast** (70%+ between figure/ground)
5. **Geometric clarity** (circles, triangles, rectangles preferred)

### Compositional Completeness
Despite emoji simplicity, each icon must:
- âœ… Communicate pillar's **primary function**
- âœ… Include pillar's **signature motif** (Y-fork, rings, grid, etc.)
- âœ… Use pillar's **color family**
- âœ… Be **visually distinct** from siblings

### Test Criteria
**"The Squint Test":** Icon must be identifiable when:
- Viewed at 16px
- Seen in peripheral vision
- Rendered in grayscale
- Displayed for <1 second
